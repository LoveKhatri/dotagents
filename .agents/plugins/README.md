# Plugins

Agent extensions that don't fit the skills/rules model — things like MCP servers, custom tools, integrations, and helper scripts.

## When to Use Plugins vs Skills vs Rules

- **Rules** — Constrain what the agent should/shouldn't do (behavioral guardrails)
- **Skills** — Teach the agent new capabilities (what it can do)
- **Plugins** — Give the agent new tools and integrations (external systems)

## Adding Plugins

Document each plugin with:
- What it does
- How to install/configure
- Which tools it works with (Claude Code, CommandCode, OpenCode)
